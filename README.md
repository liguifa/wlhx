# 大连工业大学 无机材料实验教学中心

大连工业大学无机材料实验教学中心的网站

## 功能介绍

包括教学中心的新闻、动态发布、教学文件下载、教学课题下载、学生管理、学生毕业设计选题、学生创新创业选题等.

## 项目实现

使用C#编写，主要用到.Net MVC框架，EF框架，前端使用了Easyui(登录后页面) 和 JQuery
